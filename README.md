# ☕ Java Backend Learning Journey (Core Java → Spring Boot → Spring Security)

A complete roadmap-based repository covering **Backend Development using Java**, starting from **Core Java fundamentals** to building **production-ready Spring Boot applications with Spring Security & JWT authentication**.

---

## 🚀 About This Repository

This repository is designed for:

* 🧑‍💻 Beginners starting with Java
* ⚙️ Students learning backend development
* 🎯 Developers preparing for internships / SDE roles

📌 It follows a **step-by-step structured approach**:

> Core Java → OOP → Collections → JDBC → Spring → Spring Boot → Spring Security → JWT

### 🔹 1. Core Java

* Variables, Data Types
* Loops & Conditionals
* OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)
* Exception Handling
* Multithreading

---

### 🔹 2. Collections Framework

* List, Set, Map
* HashMap, ArrayList
* Iterators & Streams

---

### 🔹 3. JDBC

* Database connection
* CRUD operations
* Prepared Statements

---

### 🔹 4. Spring Core

* Dependency Injection (DI)
* Inversion of Control (IoC)
* Bean Lifecycle

---

### 🔹 5. Spring Boot

* REST API development
* MVC Architecture
* Validation
* Global Exception Handling

### 🔹 6. Spring Data JPA

* Query optimization

* Securing APIs
